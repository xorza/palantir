Frame bench — 2026-05-20
View: 1920x2400 logical @ 2x (3840x4800 physical) so BENCH_SCALE=32
      content fits without overflow.

Arms:
  cached   = byte-identical tree every iter → Damage::Skip → renderer
             short-circuits to a backbuffer→surface tex-copy (no pass).
  partial  = same fixed target as cached; `state.tick` increments
             each iter so the footer counter text changes → small
             Partial damage rect; main render pass runs with
             LoadOp::Load + set_scissor_rect over an otherwise-static
             tree. Models the steady-state of any interactive UI
             (animating counter / blinking caret / hover highlight).
  resizing = rotates a pool of differently-sized targets so the
             backbuffer recreate escalates every frame to Full.

  *_cpu = PollType::Poll between iters → CPU pipeline only
          (record + measure + arrange + cascade + encode + compose +
          CPU side of submit), GPU runs async, not counted.
  *_gpu = PollType::Wait → full CPU + GPU frame.
